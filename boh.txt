## ✅ **13. Responsive media queries**

         Puoi adattare così:

```html
<div className="sm:px-4 max-w-[1024px] mx-auto px-2">
                                                    {/* .chat-container con padding 8px sotto i 480px */}
</div>

<h1 className="text-xl sm:text-xl text-lg">Titolo</h1>

<div className="w-8 h-8 text-xs sm:w-8 sm:h-8 sm:text-xs w-6 h-6 text-[0.6rem]">
                                                                               {/* .message-avatar */}
</div>
  ```

## ✅ Extra: definisci animazioni custom

   In `tailwind.config.js`:

```js
   extend: {
       animation: {
       fadeIn: 'fadeIn 0.3s ease-out',
       pulse: 'pulse 1.5s ease-in-out infinite',
   },
   keyframes: {
       fadeIn: {
   '0%': { opacity: '0', transform: 'translateY(10px)' },
   '100%': { opacity: '1', transform: 'translateY(0)' },
   },
   },
   }
```
